# wlib/skeleton

Démarrez votre futur projet web avec ce pack de démarrage basé sur [wlib/application](https://github.com/SamRay1024/wlib-application).

## Prérequis

- PHP 7.4.0

## Installation

```shell
composer create-project wlib/skeleton my-next-app
cd my-next-app
composer serve
```
